# Australian Ecology Field Notes

An illustrated collection of accessible, evidence-checked profiles about Australian species, habitats and ecological places.

## Profiles

- [Striped Marsh Frog](striped-marsh-frog.html)
- [Birrarung / Yarra River at Warrandyte](warrandyte-yarra-river.html)
- [Grey (or White) Mangroves around Melbourne](white-mangroves-sydney.html)

The website is made from standalone HTML pages and can be hosted directly with GitHub Pages. Each profile includes its principal information sources and image credits.

## Source files

The matching Markdown files preserve the written content in an easier-to-edit format. The HTML files are the designed web pages.
